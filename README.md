### Workflow
1. Maak een nieuwe branch voor je feature: `git checkout -b feature/jouw-feature`
2. Commit je wijzigingen: `git commit -m 'Beschrijving van wijzigingen'`
3. Push naar de branch: `git push origin feature/jouw-feature`
4. Open een Pull Request
