# Source Code

Deze map bevat alle broncode van het project.

## Structuur

Organiseer de code op een logische manier:
```
src/
├── main/           # Hoofd applicatie code
├── utils/          # Utility functies
├── models/         # Data modellen
├── controllers/    # Business logic
└── views/          # UI componenten (indien van toepassing)
```

## Richtlijnen

- Houd code modules klein en gefocust
- Gebruik duidelijke namen voor bestanden en functies
- Voeg commentaar toe bij complexe logica
- Volg de coding standards van het project
- Schrijf testbare code
